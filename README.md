
SPA-приложение для управления автопарком с возможностью просмотра, сортировки, редактирования и удаления машин, а также их отображения на карте.
# https://vehicle-test-eight.vercel.app/

## 📋 Функционал

- **Просмотр списка машин** - таблица с полями name, model, year, price
- **Создание машины** - форма для добавления нового автомобиля
- **Редактирование** - изменение name и price в модальном окне
- **Удаление** - с подтверждением действия
- **Сортировка** - по году выпуска и цене
- **Карта** - отображение всех машин по координатам (Leaflet)
- **Адаптивный дизайн** - корректное отображение на всех устройствах

## 🛠 Технологии

- **React** - библиотека для пользовательских интерфейсов
- **TypeScript** - типизация
- **Vite** - сборка проекта
- **Zustand** - управление состоянием
- **Tailwind CSS** - стилизация
- **Leaflet** - интерактивные карты
- **Axios** - HTTP-запросы
- **FSD** - архитектура Feature-Sliced Design
